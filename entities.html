Entity Codes

there are a number of characters that are reserved for HTML, which are used as the foundation of the mark-up

these symbols have entity codes associated with them


it is best practice not to use these symbols as stand alone characters direct from the keyboard; but their associated code.

this helps text editors, like vscode, compile the HTML instructions correctly; as well as the browser rendering the HTML instructions to the screen as intended.

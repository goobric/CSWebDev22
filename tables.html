Tables

what are tables?

Tables are structured sets of data, made up of rows and columns, known as a two dimensional structure. They can be a useful way of displaying data clearly.

Early Usage

In the early days of the world-wide-web, tables were commonly used to create page layouts. Now tables should only the table elements to display actual data

Lots of Elements

To create a table of data, 5-10 different table elements are available to use. These are listed in the table below

tabular data held within a table is not very accessible. Therefore, table semantic elements are used. These assist screen readers.

A list of table elements

Table Element Description
<table> top level element, similar to the <html> element in meaning, but for the table.
<thead> similar to the <header> element in meaning, but for the description of each column within the table.
<tbody> similar to the <body> element in meaning, but for the table. Which holds the main data that is being displayed.
